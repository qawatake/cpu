# ROM with My Program

## Assembly Language

1. 0000: OUT B
2. 0001: MOV A, B
3. 0010: ADD A, 0001
4. 0011: MOV B, 0000
5. 0100: ADD B, 0100
6. 0101: JNC 0100
7. 0110: MOV B, A
8. 0111: ADD A, 1100
9. 1000: JNC 0000
0. 1001: OUT 1111
1. 1010: JMP 1010
